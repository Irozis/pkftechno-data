# СС-ПОМ

Снегоочистительная машина с пневматическим обдувом. Ниже собраны данные двух изделий серии из каталога проекта и старых страниц сайта.

## Электрический шкаф А12 СС-ПОМ

Источник: https://pkftechno.ru/products/elektricheskij-shkaf-a12-ss-pom

### Текстовая информация

Является комплектующим изделием снегоочистительной машины с пневматическим обдувом СС-ПОМ и предназначен для распределения электропитания вентиляторов пневмообдува.

### Характеристики

| Параметр | Значение |
|---|---|
| Номинальное напряжение бортовой сети | 19...29 В (пост. ток) |
| Режим работы | Продолжительный |
| Климатическое исполнение У1 | по ГОСТ 15150 |
| Диапазон рабочих температур | от -40 °C до +40 °C |
| Степень защиты | IP21 по ГОСТ 14254 |
| Номинальное напряжение бортовой сети | 219...242 В (перем. ток) |
