# Ссылки для карточек ПСС-2П на Tilda

Основной сайт остаётся на Tilda. Карточки каталога на странице `pss2ptest` должны вести на универсальную страницу товара:

```text
/product?slug=<product_slug>
```

Не используйте ссылки вида `https://irozis.github.io/pkftechno-data/products/...` для пользовательских карточек.

## Важно про PP-slug

Текущие slug для шкафов коммутации ПП записаны без дополнительного дефиса после `pp`:

```text
shk-pp1-pss-2p
shk-pp2-pss-2p
shk-pp3-pss-2p
```

Их не нужно переименовывать.
